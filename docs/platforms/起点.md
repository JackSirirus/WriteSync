- **名称**：起点中文网

## 爆款规律

### 经典爆款元素

1. **系统流**：玩家心态，降低阅读门槛
2. **战神归来**：窝囊前期 + 强势反转
3. **穿越/重生**：快速建立主角金手指
4. **签到/打卡**：碎片化正反馈
5. **无敌流**：主角开局即巅峰
6. **养成流**：慢热但黏性高

### 高概率扑街元素

- 开局平淡，没有钩子
- 主角性格窝囊/圣母
- 升级节奏过慢
- 感情线拖沓
- 配角抢戏

## 禁忌/审核

### 绝对禁区（触碰即封）

| 类型 | 具体内容 | 处罚 |
|------|----------|------|
| 政治敏感 | 涉政、涉军、涉党内容；影射真实领导人；敏感历史事件 | 永久封号 |
| 色情低俗 | 露骨性描写；生殖器描写；乱伦内容；SM过度描写 | 永久封号 |
| 抄袭侵权 | 抄袭他人作品（超过20%相似度） | 永久封号+下架 |
| 违法犯罪 | 教唆犯罪、传授犯罪方法 | 永久封号 |

### 高风险内容（严格限制）

| 类型 | 具体内容 | 建议 |
|------|----------|------|
| 血腥暴力 | 过度血腥描写（断肢、内脏等）；虐待细节 | 点到为止，用"..."省略 |
| 灵异恐怖 | 封建迷信内容；鬼怪描写 | 可写"科学解释不了的现多 |
| 现实地名/人物 | 使用真实城市、医院、学校等名称 | 全部用虚构地名 |
| 同人作品 | 影视/动漫/游戏同人 | 建议原创角色，避免直接借用 |
| 感情线 | 师生恋（未成年）、一夫多妻、明a显ML | 保持尺度，可暧昧但不可露骨 |

### 审核尺度参考

| 场景 | 可写 | 不可写 |
|------|------|--------|
| 亲密接触 | 牵手、拥抱、轻吻额头 | 舌吻、脱衣、性行为描写 |
| 暴力打斗 | 打败对手、击飞 | 断肢、开膛、凌迟细节 |
| 灵异现象 | 诡异气氛、无法解释的现象 | "这是鬼"直接定性、迷信改运 |
| 主角能力 | 系统流、签到流、升级流 | 涉及现实货币的赌博系统 |
| 情感关系 | 单女主、双女主、多角恋 | 重口味的NTR、换妻 |

### 审核触发关键词（需避免）

```
敏感词类别              典型词示例
─────────────────────────────────────────
政治类              主席、总理、领导、政府、军队、警察（慎重使用）
色情低俗类          胸、臀、裸、露、揉、捏、舔（语境敏感）
暴力类              死、杀、砍、剁、凌迟、尸体（配合血腥描写）
封建迷信类          鬼、魂、诅咒、算命、风水（可用但需科学解释）
现实地名类          北京、上海、医院、学校（用虚构替代）
```

### 审核应对策略

1. **虚构地名原则**：所有地名、人名、机构名全部虚构
2. **科学解释兜底**：灵异/恐怖内容最后用"科学解释"收尾
3. **尺度模糊化**：亲密场景用"..."省略，不写细节
4. **系统替代货币**：避免"钱"直接交易，用积分/金币/贡献点
5. **配角降智处理**：反派可以坏，但不能写教唆读者做坏事

### 违规处罚流程

```
首次违规 → 警告 + 内容删除
二次违规 → 屏蔽24-72小时 + 整改通知
三次违规 → 封禁7-30天 + 作品下架
严重违规 → 永久封号 + 全平台公示
```
