# 选题检查清单

> WriteSync（共笔）项目
> 适用阶段：选题建议生成后，作为**人工参考**评估选题（v0.3.0 后选题检查 Agent 已合并入故事策划 Agent）
> 输入：选题建议列表 + 目标平台特性数据（docs/platforms/ 目录）
> 输出：每个选题的评估报告，供用户确认选题时参考
>
> 使用说明：
> - 检查前需加载 `docs/platforms/{目标平台}.md`
> - 检查结果需标注引用了平台文件的哪些维度
> - 最终选择权归用户，检查 Agent 只提风险和建议

---

## 一、平台适配性

### 1.1 题材适配

- [ ] 该题材是否属于目标平台的主流/上升题材？
- [ ] 该题材在目标平台的读者群体是否足够大？
- [ ] 该题材的订阅数据和全勤奖励是否符合平台标准？

### 1.2 风格适配

- [ ] 该题材的文风是否与目标平台调性一致？
- [ ] 更新节奏要求（字数/频率）是否与目标平台匹配？
- [ ] 目标读者的付费意愿是否与平台变现模式匹配？

### 1.3 审核适配

- [ ] 该题材是否涉及目标平台的高风险内容？
- [ ] 该题材的常见写法是否可能触碰平台审核红线？
- [ ] 该题材的受众年龄层是否与平台定位匹配？

---

## 二、市场差异化

### 2.1 题材差异化

- [ ] 该题材是否有独特的切入点（非老套的套路文）？
- [ ] 与同平台现有热门作品相比，是否有明显的差异点？
- [ ] 是纯原创还是带有同人/衍生性质（后者有版权风险）？

### 2.2 卖点差异化

- [ ] 核心卖点是否足够独特（非"升级打怪"等泛泛卖点）？
- [ ] 卖点是否能在开篇三章内展现（读者等不了太久）？
- [ ] 卖点是否具有可持续性（能否支撑 30-50 万字甚至更长）？

### 2.3 竞品对比

- [ ] 与同平台同题材的热门作品相比，自家选题的竞争优势在哪里？
- [ ] 竞品的不足是否恰好是自家选题的发力点？
- [ ] 是否存在竞品已把市场做烂的风险（读者审美疲劳）？

---

## 三、题材饱和度

### 3.1 题材热度评估

| 热度等级 | 描述 | 风险 |
|----------|------|------|
| ★★★★★ 热门 | 流量大但竞争激烈 | 头部效应明显，新书难出头 |
| ★★★★☆ 上升 | 近期增速明显 | 有机会，但需差异化 |
| ★★★☆☆ 平稳 | 读者稳定，竞争一般 | 稳妥但天花板有限 |
| ★★☆☆☆ 下降 | 热度在衰退 | 市场萎缩中 |
| ★☆☆☆☆ 冷门 | 流量很小 | 除非有特殊卖点，否则慎选 |

- [ ] 该题材当前处于哪个热度等级？
- [ ] 热度趋势是上升还是下降？
- [ ] 热度等级与自身差异化能力是否匹配？

### 3.2 细分题材评估

- [ ] 是否选择了过于宽泛的题材分类（"都市"太大，"都市鉴宝"更细分）？
- [ ] 细分题材是否有足够读者群体？
- [ ] 细分题材是否有可拓展性（能否写到后期不撞墙）？

---

## 四、核心卖点评估

### 4.1 卖点清晰度

- [ ] 一句话能否说清这个选题的核心卖点？
- [ ] 卖点是否具体而非泛泛（"爽文"不如"主角靠系统面板一眼看穿敌人弱点然后针对性反杀"具体）？
- [ ] 卖点是否对目标读者有足够吸引力？

### 4.2 卖点可执行性

- [ ] 卖点是否有内在逻辑支撑（不是作者想写哪写哪）？
- [ ] 卖点能否转化为可持续的情节动力（不是三章就写完了）？
- [ ] 该卖点是否适合 3000-5000 字/天的日更节奏？

### 4.3 卖点持久性

- [ ] 该卖点能支撑多少字的故事（5万/20万/50万/100万）？
- [ ] 卖点是否有递进结构（前期爽完中期还有什么新的爽点）？
- [ ] 是否存在"后期无话可写"的风险？

---

## 五、高概率扑街预警

### 5.1 题材扑街预警

- [ ] 是否选择了已泛滥的套路题材（都市战神/传统玄幻/同质穿越）？
- [ ] 是否选择了有政策风险的题材（现实军旅/现实官场）？
- [ ] 是否选择了版权敏感的同人题材？

### 5.2 人设扑街预警

- [ ] 主角人设是否涉及常见毒点（圣母/窝囊/优柔寡断）？
- [ ] 主角目标是否清晰且有驱动力（没有目标的角色走不远）？
- [ ] 主角与读者共情路径是否建立（读者能代入主角）？

### 5.3 节奏扑街预警

- [ ] 卖点是否需要很长的前期铺垫才能展开（读者会中途弃书）？
- [ ] 开篇前3章的钩子设计是否有初步思路？

> 注：开篇钩子的具体设计在后续策划/章纲阶段进行，选题阶段只需确认是否有初步思路。

### 5.4 市场扑街预警

- [ ] 该选题是否依赖特殊的市场窗口（等某个热点）？
- [ ] 该选题是否对读者文化水平有较高要求（降低受众面）？
- [ ] 该选题是否有明显的变现天花板（全勤可以，但长期收益有限）？

- 风格适配：⭐⭐⭐⭐⭐ / 🔴 不适配

```markdown

| 评估维度 | 选题A | 选题B | 选题C |
|----------|-------|-------|-------|
| 题材适配 | ⭐⭐⭐⭐ | ⭐⭐⭐⭐⭐ | ⭐⭐⭐ |

| 题材热度 | 热门 | 上升 | 平稳 |
| 卖点清晰度 | ⭐⭐⭐⭐⭐ | ⭐⭐⭐ | ⭐⭐⭐⭐ |

| **综合推荐** | **★★★** | **★★★★★** | **★★** |

### 推荐结论
[推荐哪个选题，或建议如何组合/修改]
```

## 八、特殊说明

1. **选题检查是辅助决策**，最终选择权在用户手中
2. **检查 Agent 提风险，不替用户做决定**
3. **高风险不等于不能写**，有些高风险题材反而因为竞争少而容易出圈
4. **平台特性数据以 docs/platforms/ 各平台文件为准**，检查 Agent 引用时需标注来源平台文件
5. **选题阶段只做初步判断**，具体执行层面的检查（如开篇钩子、节奏设计）在后续阶段进行
