### 设计哲学约束

1. **不破坏主 Agent 编排**：新增功能是 Agent 能力的增强，不替代主 Agent 决策
2. **用户可选择性使用**：新功能是可选的增强，非强制流程
3. **保持架构简洁**：7 Agent + 纯 Python 循环，不膨胀到 StoryForge 的 30+ 适配器
4. **向后兼容**：所有变更不破坏现有项目数据和 API —— **每个 Phase 有专用兼容性验收项**
