# 钩子技法

> WriteSync（共笔）项目
> 用途：帮助策划 Agent / 章纲 Agent / 文笔 Agent 设计章节钩子的核心技法参考
> 适用阶段：章纲阶段规划各章钩子，文笔阶段执行钩子写法
> 核心目标：让读者每章结束时产生"必须看下一章"的冲动

---

## 一、钩子的本质

### 1.1 什么是钩子

钩子是章节结尾的**悬念装置**，其本质是**打断读者的满足感**，制造"未完成"的张力。

读者在阅读过程中积累了一定的情绪（好奇/紧张/期待），然后在章节末尾被打断——这种"被打断的不满足感"驱动读者继续读下去。

```
情绪积累 → 情绪接近顶点 → 章节断章 → 读者：不行，我要继续看！
```

### 1.2 钩子与断章的关系

钩子是**内容设计**，断章是**技术实现**。

- 钩子决定"用什么悬念吸引读者"
- 断章决定"在哪个位置打断读者"

好的钩子 + 好的断章 = 读者追读。

### 1.3 钩子的三个层次

| 层次 | 说明 | 示例 |
|------|------|------|
| **悬念钩子** | 留下未解的问题 | 门后是谁？|
| **冲突钩子** | 冲突即将爆发 | 敌人已经动手了 |
| **期待钩子** | 预告即将发生的好事 | 三天后就是大比 |
| **危机钩子** | 危险正在逼近 | 黑影已经出现在窗外 |
| **反转钩子** | 揭露关键信息 | 原来这一切都是他设计的 |

### 1.4 钩子的强度分级

| 强度 | 效果 | 适用场景 |
|------|------|----------|
| ⭐ 弱钩子 | 读者可能选择跳过 | 平缓叙述结尾、情绪平淡处 |
| ⭐⭐ 中钩子 | 读者可能会继续看 | 日常过渡、情感铺垫 |
| ⭐⭐⭐ 强钩子 | 读者想继续看 | 冲突升级、悬念加深 |
| ⭐⭐⭐⭐ 强钩子 | 读者必须继续看 | 危机来临、关键信息即将揭露 |
| ⭐⭐⭐⭐⭐ 极强钩子 | 读者无法停止 | 生死危机、重大反转、命运抉择 |

## 二、钩子设计原则

### 2.1 钩子必须在情绪高点断章

断章的最佳位置是**情绪顶点稍前**，不是情绪顶点，也不是情绪顶点之后。

```
情绪强度 ↑
         |    ★（断章点：情绪即将爆发的前一刻）
         |   /
         |  /
         | ★
         |/
         └────────────────────────────→ 时间
```

**为什么不能在顶点断章？**
- 顶点断章，读者已经爽完/紧张完，满足感够了，可能停止
- 读者自己脑补了结果，没有"必须知道"的动力

**为什么不能在顶点之前断章？**
- 铺垫还不够，情绪还没到位
- 读者还在"等"，不会急着看

### 2.2 钩子必须在章节末尾最后 100-300 字内

钩子不能埋太深。如果在章节开头就埋下钩子，等到结尾才呼应，读者可能已经忘了。

**原则**：悬念越强，距离断章点越近越好。

| 钩子强度 | 建议位置（距章末）|
|----------|------------------|
| 弱钩子 | 300-500 字 |
| 中钩子 | 200-300 字 |
| 强钩子 | 100-200 字 |
| 极强钩子 | 50-100 字 |

### 2.3 钩子必须与下一章开头衔接

章节之间不是孤立的。钩子和下一章开头要有自然的衔接。

```
第N章结尾钩子：黑影正在逼近……
第N+1章开头：不是黑影，是……（回应钩子）
```

**常见错误**：章末说A，章首从B开始，读者感到困惑或被欺骗。

### 2.4 钩子必须足够具体

泛泛的钩子没有吸引力。

| ❌ 泛泛的钩子 | ✅ 具体的钩子 |
|--------------|--------------|
| "还有更精彩的故事等着你" | "他低头看向胸口，那里插着一柄短剑" |
| "更大的危险即将来临" | "就在他转身的瞬间，袖口滑落一张纸条，上面写着……" |
| "秘密即将被揭开" | "系统面板突然闪烁，一行从未出现过的文字浮现：'检测到宿主触碰禁忌领域'" |

## 三、钩子类型详解

### 3.1 悬念钩子

通过未解的问题吸引读者。

**写法**：
```
- 门后传来脚步声
- 他突然发现手中的玉佩正在发光
- 系统面板弹出一行从未见过的提示
```

**要点**：
- 悬念要具体（是什么/谁/为什么）
- 悬念要在下一章尽快回应（不要吊超过3章）
- 悬念要与主线相关

### 3.2 冲突钩子

通过即将爆发的冲突吸引读者。

**写法**：
```
- 话音未落，门外传来急促的敲门声
- 他刚转身，一道剑气已经刺向他的后背
- 就在他准备离开时，苏长老突然开口："站住"
```

**要点**：
- 冲突要立即发生（下一章就要处理）
- 冲突的后果要严重（让读者担心）
- 冲突要有悬念（读者想看主角怎么应对）

### 3.3 期待钩子

通过预告即将发生的正面事件吸引读者。

**写法**：
```
- 三日后，便是宗门大比之日
- 据说这个遗迹中，藏有上古大能留下的传承
- 三个月后，他就会回到这座城市
```

**要点**：
- 期待的事件要足够有吸引力
- 要给读者一个"等"的预期
- 要在承诺的时间点兑现（不能无限拖延）

### 3.4 危机钩子

通过即将到来的危险吸引读者。

**写法**：
```
- 远处，一道黑影正在逼近
- 计时器显示：还剩三分钟
- 他突然意识到，自己已经中毒
```

**要点**：
- 危机要有紧迫感（时间限制）
- 危机要有严重后果（主角可能死/失败/失去）
- 要展示主角有应对危机的可能性（读者才会期待）

### 3.5 反转钩子

通过揭露关键信息制造反差吸引读者。

**写法**：
```
- 原来这一切，都是他设计的
- 他低头看向自己的手，发现竟然变成了透明
- 当他转过身，才看清那张脸——竟然是他
```

**要点**：
- 反转要出乎意料但又在情理之中
- 反转要改变读者对之前情节的认知
- 反转要影响主角接下来的决策

### 3.6 情感钩子

通过情感悬念吸引读者。

**写法**：
```
- 她在信中只写了一句话：我等你回来
- 当他翻开相册，看到最后一张照片时，愣住了
- 她的眼眶红了，却没有说一句话
```

**要点**：
- 情感要真实动人
- 情感要有未完成感
- 要在下一章或适当时候回应

## 四、不同节奏类型的钩子设计

### 4.1 爆发型章节的钩子

爆发型章节的特点是情绪逐渐推向高点。

**钩子类型**：强钩子 / 极强钩子
**断章位置**：情绪顶点前100-200字
**钩子示例**：危机钩子、反转钩子

```
写法：
铺垫（前80%）→ 情绪逐渐推高 → 钩子（后20%，断章点）
→ 情绪即将爆发前断章，读者自己想象爆发结果
```

### 4.2 积蓄型章节的钩子

积蓄型章节的特点是缓慢积累情绪，为下一章的爆发做铺垫。

**钩子类型**：期待钩子、悬念钩子
**断章位置**：情绪积累的某个节点
**钩子示例**：预告即将到来的事件

```
写法：
日常/铺垫（前70%）→ 发现异常/埋下伏笔（中20%）→ 钩子（后10%）
→ 钩子指向下一章的爆发
```

### 4.3 交替型章节的钩子

交替型章节的特点是有张有弛。

**钩子类型**：根据章节末尾的情绪状态选择
**断章位置**：张力最大处
**钩子示例**：冲突钩子（如果张→弛）或悬念钩子（如果弛→张）

```
写法：
张（前40%）→ 弛（中20%）→ 张（后40%）→ 钩子（断章）
→ 在第二次张力高点断章
```

### 4.4 悬念型章节的钩子

悬念型章节的特点是持续紧张，不释放。

**钩子类型**：危机钩子
**断章位置**：危机加剧的过程中
**钩子示例**：危险即将发生

```
写法：
危机开始（前30%）→ 危机升级（中40%）→ 危机即将爆发（后30%）
→ 在危机即将爆发的瞬间断章
```

## 五、全书钩子节奏规划

### 5.1 钩子强度曲线

```
钩子强度 ↑
         |      ★            ★            ★
         |  ★        ★  ★        ★  ★
         |___________________________________→ 章节
         0   10   20   30   40   50   60
```

| 阶段 | 钩子强度 | 说明 |
|------|----------|------|
| 开局期（1-10章）| 强 | 每章必须有强钩子，建立追读习惯 |
| 发展期（10-40章）| 中-强 | 每2-3章有一个强钩子 |
| 高潮期（40-60章）| 极强 | 每章都有强钩子，持续紧张 |
| 收尾期（60章+）| 中 | 钩子减少，叙事为主 |

### 5.2 钩子类型分布

不要每章都用同一种钩子，要多样化。

| 钩子类型 | 出现频率建议 | 说明 |
|----------|-------------|------|
| 悬念钩子 | 30% | 最通用 |
| 冲突钩子 | 25% | 最紧张 |
| 期待钩子 | 20% | 最正面 |
| 危机钩子 | 15% | 最紧迫 |
| 反转钩子 | 7% | 最震撼（不能滥用）|
| 情感钩子 | 3% | 最动人（偶尔使用）|

## 六、钩子设计的常见错误

### 错误1：钩子太泛泛

**问题**："还有更精彩的故事"、"更大的挑战即将来临"，读者不知道具体是什么。

**对策**：钩子要具体，要让读者知道"是什么即将发生"。

### 错误2：钩子埋太深

**问题**：章节开头就埋下钩子，读者到结尾已经忘了。

**对策**：钩子要在章节末尾最后100-300字内。越强的钩子，越要靠近断章点。

### 错误3：钩子不兑现

**问题**：章末说"更大的危险即将来临"，10章后还没发生。

**对策**：钩子承诺的悬念要在3章内兑现。如果需要更长时间，要在中途有小的回应/进展。

### 错误4：钩子与章节无关

**问题**：章节讲的是A，钩子突然扯到B，两边都没有关联。

**对策**：钩子要自然地从章节内容中生长出来，是章节发展的自然延伸，不是强行插入。

### 错误5：连续弱钩子

**问题**：连续3-5章都是弱钩子，读者失去耐心。

**对策**：每3章至少有一个强钩子，维持读者的追读动力。

## 七、与 Agent 的配合

### 策划 Agent / 章纲 Agent 使用指南
- 在章纲阶段规划各章的钩子类型和强度
- 参考"全书钩子节奏规划"设计钩子分布
- 确保钩子类型多样化
- 钩子要与下一章内容有衔接规划

### 文笔 Agent 使用指南
- 参考"钩子类型详解"选择合适的钩子写法
- 遵循"钩子设计原则"
- 避免"常见错误"

### 节奏 Agent 使用指南
- 检查章节结尾是否有钩子
- 检查钩子强度是否与章节节奏类型匹配
- 检查断章位置是否合适（情绪高点前100-300字）

## 八、开篇钩子与章节钩子对照

本文是**章节钩子的通用技法**，《开篇三章技法》是**开篇钩子的专项设计**。两者维度不同，用法有别：

| 开篇钩子类型（开篇三章）| 章节钩子类型（本文）| 对应关系 |
|------------------------|---------------------|----------|
| 冲突型 | 冲突钩子 | 开篇用冲突场景开场 = 章节用冲突悬念断章 |
| 悬念型 | 悬念钩子 | 一致 |
| 反差型 | 反转钩子 | 反差开篇 ≈ 反转断章（但时机不同）|
| 金手指型 | 期待钩子 | 金手指开篇预告能力 → 期待钩子预告事件 |
| 爽点型 | 期待钩子 | 爽点开篇 ≈ 期待钩子（都有"预告"性质）|
| 情感型 | 情感钩子 | 一致 |

> 核心区别：开篇钩子决定"用什么方式开场吸引读者"，章节钩子决定"在什么位置用什么悬念断章"。两者配合使用——开篇三章用"开篇钩子类型"设计前三章，章纲/文笔用"章节钩子类型"设计各章断章。
