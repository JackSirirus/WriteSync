# 全书审查维度

1. **结构**：三幕节奏是否合理、高潮位置是否恰当、整体叙事是否完整
2. **角色弧线**：每个角色的变化轨迹是否连贯、是否有断裂或跳跃
3. **伏笔**：伏笔是否有植有收、是否有遗漏未收的伏线
4. **节奏**：全书节奏是否有起伏、是否有大段枯燥或推进过快
5. **篇幅**：各章字数分配是否合理、是否有严重注水或过短
6. **世界观一致性**：设定使用是否前后一致、是否有吃书
7. **主题**：核心主题是否贯穿全书、结局是否回应主题
